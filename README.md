# Features
+ 支持普通数据库和加密数据库
+ 支持预定义表和动态生成表
+ 支持动态扩展表(目前只支持新增字段)
+ 记录当前数据库版本号,升级逻辑需自行实现
+ 支持sync 和 async I/O
